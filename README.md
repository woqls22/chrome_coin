# Chrome Coin
- 크롬 익스텐션 : 상사몰래 코인하기
- 실시간 코인 가격 확인 애플리케이션 ( From Upbit API )
- 즐겨찾기 기능 추가 예정

## Install
```
$ yarn install
$ yarn start
```
